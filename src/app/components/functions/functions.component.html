Apuntes de funciones

¿Qué son las funciones?

Las funciones son un método para encapsular un fragmento de código

Tipos de funcinoes

  • Según la sincronía:

    Síncronas o asíncronas

    1. {{ " `function asyncFunction async() { setTimeOut(console.log('esto es una funcion asíncrona'),1000) } " }}
    2. {{ "function syncFunction() { console.log('esto es un funcion síncrona') } " }}
  • Según como estén escritas:

    Normales o tipo flecha

    1. {{ " const arrowFunction = () => { console.log('funcion tipo flecha') }" }}
    2. {{ "function commonFunction () { console.log('function común') }" }}

    • Muy importante: las funciones tienen que ser invocadas para que se ejecuten.

Código encapsulado

El código que sucede dentro de una función no tiene comunicación con el exterior, por lo tanto los datos que queramos utilizar fuera de una función se incluiran en el return, y los que queramos que utilice la función habrá que introducírselos a través de argumentos, Ejemplo: {{ "function functionSuma (argumento1, argumento2) { const suma = argumento2 + argumento2; return suma; }" }} {{ "functionSuma(5,7);" }}
